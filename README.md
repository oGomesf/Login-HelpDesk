Projeto pessoal de um Sistema de Chamados (HELP DESK) integrado com um sistema de Login.

-- OBJETIVOS --

1- Autenticar usuário e senha ou cadastrar um novo caso nao possua.
2- Inicializar o Dashboard após autenticaçao bem sucedida
3- Listar Chamados em abertos por prioridade e possibilitar uma busca
4- Adicionar, modificar e remover chamados em aberto
5- atualizar status
6- logout

Stack escolhida: Python, SQLite3, CustomTKinter e TKinter
